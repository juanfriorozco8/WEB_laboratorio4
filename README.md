## Descripción

Disci-App es una aplicación web de gamificación enfocada en reforzar buenos hábitos mediante un sistema de misiones y recompensas de XP. El usuario puede crear misiones con distintos niveles de dificultad, completarlas y ver cómo su rango sube conforme acumula experiencia.

## Funcionalidades

- Crear misiones con nombre, descripción y dificultad
- Sistema de XP: Fácil (10 XP), Normal (25 XP), Difícil (50 XP)
- XP global acumulado visible en el header
- 3 rangos según XP: Novice (0–99), Warrior (100–299), Champion (300+)
- Barra de progreso de XP
- Marcar misiones como completadas → estado `SUCCESFUL`
- `console.log` por cada misión creada para validar su estructura

## Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)

Sin librerías ni frameworks externos.
